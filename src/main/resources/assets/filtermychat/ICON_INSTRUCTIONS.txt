ИНСТРУКЦИИ ПО СОЗДАНИЮ ИКОНКИ МОДА

Для отображения иконки в ModMenu необходимо:

1. Создать PNG файл размером 128x128 пикселей или больше (рекомендуется 256x256)
2. Заменить файл icon.png в этой папке на вашу иконку
3. Убедиться, что файл имеет расширение .png

Требования к иконке:
- Формат: PNG
- Размер: минимум 128x128 пикселей
- Файл должен быть меньше 1MB
- Прозрачный фон (опционально)

Примеры программ для создания иконки:
- Paint.NET (бесплатно)
- GIMP (бесплатно)
- Photoshop
- Canva (онлайн)

После замены иконки пересоберите мод командой: .\gradlew remapJar
